## Como trabalhamos

```
Discovery  →  Design  →  Build  →  Deploy  →  Observe
  código      ADRs     PRs     health-     métricas,
  lido     docs    revisados   gate +      traces, SLOs
                              rollback
```

1. **Discovery** — leitura de código, arquitetura, métricas existentes
2. **Design** — decisões documentadas (ADRs), trade-offs explícitos
3. **Build** — código em pair/mob, PRs revisados, CI rigoroso
4. **Deploy** — rollout com health-gate e rollback automatizado
5. **Observe** — métricas, traces, SLOs monitorados desde o dia 1
